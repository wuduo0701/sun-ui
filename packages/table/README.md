# @sum-ui/table

## Install

```bash
npm i @sum-ui/table
# or
yarn add @sum-ui/table
```

## Usage

```vue
<template>

<script lang="ts">
import { defineComponent, toRefs } from 'vue'

export default defineComponent({

    setup() {}
})
</script>
```
