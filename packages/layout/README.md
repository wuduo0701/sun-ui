# @sum-ui/layout

## Install

```bash
npm i @sum-ui/layout
# or
yarn add @sum-ui/layout
```

## Usage

```vue
<template>
    <sum-layout />
</template>

<script lang="ts">
import { defineComponent, toRefs } from 'vue'
import SumLayout from '@sum-ui/layout'

export default defineComponent({
    components: { SumLayout },
    setup() {}
})
</script>
```
